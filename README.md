# Remax Plugins

Remax 官方插件库。

## 插件列表

|                       |                       |
|-----------------------|-----------------------|
| [less](packages/less) | Less 支持               |
| [sass](packages/sass) | Sass 支持               |
| [stylus](packages/stylus) | Stylus 支持             |
| [linaria](packages/linaria) | Linaria 支持            |
| [wx-shake](packages/wx-shake) | 移除微信 wx api 接口        |
