# @remax/plugin-stylus

增加 Stylus 支持。

## 安装

```bash
$ npm install @remax/plugin-stylus --save
```

## 使用

```js
const stylus = require('@remax/plugin-stylus');

module.exports = {
  plugins: [stylus()],
};
```

## 配置

同 `stylus-loader` 的[配置](https://github.com/shama/stylus-loader)。
