# @remax/plugin-less

增加 Less 支持。

## 安装

```bash
$ npm install @remax/plugin-less --save
```

## 使用

```js
const less = require('@remax/plugin-less');

module.exports = {
  plugins: [less()],
};
```

## 配置

同 `less-loader` 的[配置](https://github.com/webpack-contrib/less-loader#options)。
